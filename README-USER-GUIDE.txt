README USER GUIDE - Smart Notes
=====================================

Overview
--------
Smart Notes is a study productivity web app with notes, tasks, sharing, notifications, and an AI assistant (Smart Notes AI). It helps you organize study material, track deadlines, and collaborate.

Getting Started
--------------
1. Open the app in your browser.
2. Create an account or sign in.
3. Use the top-left menu button (three lines) to open navigation.

Navigation
----------
Open the sidebar menu to access:
- Home
- Notes
- Tasks
- Smart Notes AI
- Shares
- Notifications
- Account

Home
----
- Overview panel and quick access to features.
- Recommended entry point after sign-in.

Notes
-----
- Create, edit, and delete study notes.
- Search and filter notes.
- Generate AI help on notes (summaries, explanations).
- Share notes with other users using share links.

Tasks
-----
- Create tasks with due dates.
- Track progress and mark tasks complete.
- Share tasks with collaborators using share links.

Smart Notes AI (AI Assistant)
---------------------
- Ask questions, summarize, or get explanations.
- Modes:
  - General: open Q&A.
  - Study: learning help and explanations.
  - Project: guided or fast help on projects.
- Chat history is saved per session.
- You can share chat sessions using share links.

Notifications
-------------
- View notifications in a dedicated page.
- Notifications include task due reminders, share invites, and study reminders.
- Mark all as read or clear all.
- Notifications are ordered newest first and include date/time.

Shares
------
- Manage shared links for notes, tasks, and chats.
- Invite users to collaborate or revoke links.

Account
-------
- Update account settings.
- Reset password.
- Delete account (permanent).

Security Notes
--------------
- Use a strong password.
- Log out on shared devices.
- Password reset emails are only sent if an account exists for the email.

Troubleshooting
---------------
- If you can’t sign in, verify username/password and check internet connection.
- If you forgot your password, use “Forgot password?” on the login screen.
- If data appears missing, refresh the page and ensure you’re signed in.

Support
-------
If you need help, contact your administrator or deployment owner.
